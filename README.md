## NYSMC
* 选择加载“壳应用”/“真应用”的简单框架。
* Choose Sheel/Application framework.

## <a id="How_to_use_NYSMC"></a>How to use NYSMC
* Installation with CocoaPods：`pod 'NYSMC'`
* Manual import：
    * Drag All files in the `NYSMC` folder to project
    * Import the main file：`#import "NYSCake.h"`
